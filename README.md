# Nothing Prayer App Documentation

## New Glyph Interface Features

The Glyph interface introduces an innovative way to interact with the prayer app through visual glyph patterns that enhance usability.

## Nothing Phone 3 Rear Display Support

The app is optimized to support the unique rear display of the Nothing Phone 3, offering enhanced notifications and features.

## How Glyph Patterns Work for Each Prayer

Glyph patterns serve as visual cues for each prayer time. These patterns light up to indicate the beginning and end of prayer times, helping users manage their schedules effectively.

## Installation Instructions

To install the Nothing Prayer App on your device:
1. Download the app from the official store.
2. Follow the on-screen instructions to complete the setup.
3. Ensure your device supports Glyph functionality.

## Supported Devices

- Nothing Phone 1
- Nothing Phone 2
- Nothing Phone 2a
- Nothing Phone 3

## Feature List

- 5-minute prayer warnings before each prayer time.
- Dynamic Glyph pattern notifications.
- Customizable settings for individual user preferences.

## How to Toggle Glyph On/Off

Users can enable or disable the Glyph feature in the app settings under "Glyph Options." Simply toggle the switch to activate or deactivate.

## Screenshots Description of Front and Rear Displays

- **Front Display:** Shows the current prayer time and next prayer warning.
- **Rear Display:** Displays the glyph patterns associated with the current prayer time, providing a visual reminder for the user.
